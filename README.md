# 🏆 AntiDoping Quest

An interactive web application that educates users about anti-doping through quizzes, challenges, rewards, and AI-powered assistance.

## 🚀 Features

- 🔐 User Authentication (Login & Register)
- 🤖 AI Chat Assistant
- 📝 Interactive Quizzes
- 📊 Progress Analytics
- 🏅 Badges & XP System
- 🏆 Leaderboard
- 🎓 Certificate Generation
- 👨‍💼 Admin Dashboard
- 📱 Responsive Design

## 🛠️ Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL
- Prisma ORM

### Authentication
- JWT
- bcrypt

## 📦 Installation

### Clone the repository

```bash
git clone https://github.com/Yasmitha28/antidoping-quest.git
```

### Install dependencies

```bash
npm install
```

### Start the frontend

```bash
cd client
npm install
npm run dev
```

### Start the backend

```bash
cd server
npm install
npm start
```

## 🎯 Objective

The main objective of this project is to spread awareness about anti-doping rules through an engaging gamified learning platform where users can improve their knowledge while earning rewards and tracking their progress.
